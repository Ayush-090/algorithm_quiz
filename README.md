An interactive Algorithm Quiz Web App with adaptive difficulty and ELO-based rating system, built for mastering DAA (Design and Analysis of Algorithms).

✨ Features


🧠 Adaptive Question Difficulty (AI-like behavior)


📊 ELO Rating System (Skill tracking)


🎯 Topic-based Questions:


Dynamic Programming


Greedy Algorithms


Graph Algorithms




🔥 Streak System (Motivation booster)


📈 Real-time Progress Tracking


💡 Instant Answer Explanations


🏆 Local Leaderboard (Top players stored in browser)


⚡ Smooth UI + Animations

View leaderboard
